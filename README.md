# magenta-orb-desktop

Прозрачное desktop-приложение с процедурной анимацией orb на `Electron + Three.js`.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Управление

- `Esc` закрывает окно
- `Cmd+Shift+T` переключает режим поверх всех окон

## Примечание

Если зависимости ещё не установлены, сначала выполни `npm install`, а затем `npm start`.
